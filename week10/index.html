Validating Forms

Before submitting data to the server, it is important to ensure all required form controls are filled out, in the correct format. This is called client-side form validation,
and helps ensure data submitted matches the requirements set forth in the various form controls.

Client-side validation

Client-side validation is an initial check and an important feature of good user experience; by catching invalid data on the client-side, the user can fix it straight away.

There are two different types of client-side validation

  • Built-in form validation(uses HTML5)
  • JavaScript validation

Using Fetch
